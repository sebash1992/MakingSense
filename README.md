# My Blog

My Blog is a small application writed using .net Core 2.0 and Angular, that shows the basic action that can be performed in a blog

## Installation



## Usage

```cmd
npm install
cd application.SPA
ng build --prod
cd ..\application.API\
dotnet run
```
